CRUD API Documentation

Base URL: https://cs.stcc.edu/csc114/api

Note: All endpoints require an Authorization header for Basic Authentication. Please contact Tony Silvestri for the login information.

1. Get All Addresses

Fetch all addresses with optional pagination.

Endpoint: GET /addresses

Query Parameters:

  • page (optional) - Page number (default: 1)
  • limit (optional) - Items per page (default: 10)

Sample URL:

https://cs.stcc.edu/csc114/api/addresses?page=1&limit=5

2. Get Address by ID

Retrieve a single address by its unique ID.

Endpoint: GET /addresses/:id

Sample URL:

https://cs.stcc.edu/csc114/api/addresses/1

3. Create a New Address

Add a new address record to the database.

Endpoint: POST /addresses

Request Body: (JSON)


{
    "name": "John Doe",
    "address": "123 Main St",
    "city": "Springfield",
    "state": "IL",
    "zip": "62704"
}

4. Update an Address

Update an existing address by its ID.

Endpoint: PUT /addresses/:id

Request Body: (JSON)


{
    "name": "John Smith",
    "address": "456 Elm St",
    "city": "Columbus",
    "state": "OH",
    "zip": "43085"
}

Sample URL:

https://cs.stcc.edu/csc114/api/addresses/1

5. Delete an Address

Remove an address by its ID.

Endpoint: DELETE /addresses/:id

Sample URL:

https://cs.stcc.edu/csc114/api/addresses/1

6. Search for Addresses

Search all fields for a specific query string.

Endpoint: GET /addresses/search

Query Parameters:

  • query (required) - The string to search for

Sample URL:

https://cs.stcc.edu/csc114/api/addresses/search?query=John Main St

7. Get Address Count

Get the total count of address records.

Endpoint: GET /addresses/count

Sample URL:

https://cs.stcc.edu/csc114/api/addresses/count
